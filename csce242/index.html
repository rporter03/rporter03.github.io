CSCE 242: Website Design and Development

This class is about the fundamentals of building a website which aims to familiarize students with essential tools such as HTML, CSS, and JavaScript.

Assignments

  1. Assignment 1 - Basic HTML

    This assignment's goal was to get familiarized with the basics of HTML.

  2. Assignment 2 - Basic CSS

    This assignment showcases basic styling using CSS.

  3. Assignment 3 - Page Layout

    This assignment showcases and familiarizes the use of flex in CSS.

  1. Assignment 4 - Recreate CSS Page

    This assignment's goal was to recreate one of USC's webpages using flex and CSS.

  2. Assignment 5 - Introduction to JavaScript

    This assignment aimed to teach the basic conventions of JavaScript.

  3. Assignment 6 - JavaScript Conditionals

    This assignment showcases the use of JavaScript conditionals, like if-else statements.

  1. Assignment 7 - JavaScript Loops

    This assignment uses JavaScript loops to create a scene of sun/moon, clouds, and trees.

  2. Assignment 8 - JavaScript Arrays

    This assignment showcases the use of JavaScript arrays to load various images onto a website.

Projects

  2. Project Part 2 - Wireframes

  3. Project Part 3 - HTML and CSS

    This part of the project takes the layout from the wireframe and implements it into HTML and CSS.

  1. Project Part 4 - HTML and CSS

    This step of the project adds actual text, color, and images to the previous implementation of the website.

  2. Project Part 5 - Improve Your Website
